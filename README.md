# Treehouse Website

A basic website describing Treehouse, a parental control application for managing children's screen time and promoting healthy device habits.

## Treehouse App Description

Treehouse is a parental control application that helps parents manage their children's screen time and promote healthy device habits. Parents can connect their child's device, view usage insights, and set appropriate limits to encourage balanced technology use. The app also motivates children to spend less time on their devices through optional in-app games and progression systems designed to reward healthy behavior. Treehouse aims to support digital wellbeing while giving parents clear visibility and control over how their children interact with apps and devices.

## Features

- Device connection and monitoring
- Usage insights and reports
- Customizable time limits
- Motivational games and rewards
- Focus on digital wellbeing
